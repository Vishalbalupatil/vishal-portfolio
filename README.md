# Vishal Patil Portfolio

Futuristic personal site built with **Next.js 14**, **TypeScript**, **Tailwind CSS**, **Framer Motion** and **shadcn/ui**.

## Getting Started

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm start
```

## Testing

```bash
npm test       # unit tests with Vitest
npm run e2e    # Playwright smoke tests
```

## Deploy

1. Push this repository to GitHub.
2. Create a new project on [Vercel](https://vercel.com) and import the repo.
3. Set the framework preset to **Next.js** and deploy.

The site includes automatic sitemap and robots generation, SEO metadata and consent-based Vercel Analytics.
